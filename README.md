# Task 14 - Prometheus & Grafana Setup

## Steps I followed
1. Installed Docker & Docker Compose
2. Created docker-compose.yml for Prometheus and Grafana
3. Started services using docker compose up -d
4. Opened Prometheus on http://localhost:9090
5. Opened Grafana on http://localhost:3001
6. Added Prometheus as Data Source in Grafana
7. Verified metrics in Explore section
